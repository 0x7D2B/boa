Checks if the lexical declaration continues with more bindings.

If it does, it will advance the internal cursor to the next identifier token. A Lexical Declaration continues its binding list if we find a , character. A New line indicates the same as a ;.

More information: https://tc39.es/ecma262/#prod-LexicalDeclaration.
